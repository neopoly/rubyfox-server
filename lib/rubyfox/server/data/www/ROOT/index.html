» Administration

Launch the Administration Tool 2X (default username & password: sfsadmin).

PLEASE NOTE: if you are installing SFS2X on a public server, make sure to immediately change the default Admin credentials using a secure password. To do so, login in the AdminTool, select Server Settings > Remote Admin > and apply the changes. Then restart SmartFoxServer.
